大樂透規則

請參閱 大樂透官網規則說明頁面。

介面

輸入

日期輸入格式為: YYYY / MM ，模式欄位說明於下方。
按 查詢 ，會將資料顯示於底下名稱為search資料上。
按 加入 ，可將查詢出來的資料以日期和模式為名稱顯示於圖表。
按 重設 ，可將圖表清除。

圖表

點選圖表上方資料名稱可以取消顯示此筆資料。
將滑鼠停在圖上可以顯示詳細資料。

模式

模式一：　普通獎號出現次數查詢

在模式欄位輸入 all 可以查詢各獎號出現在普通獎號的次數。

模式二：　特別獎號出現次數查詢

在模式欄位輸入 sp 可以查詢各獎號出現在特別獎號的次數。

模式三：　相連號碼次數查詢

在模式欄位輸入 欲查詢數字 可以查詢同時與此號碼出現的其他號碼出現次數。

實作方法

抓取資料

使用 node.js 定時抓取 樂透雲 獎號資料，按日期排列後轉換成jSON格式。

整理資料

使用 javascript 讀取jSON，並讀取使用者輸入的日期、模式，找出對應的資料。

顯示資料

使用 chart.js ，將對應資料用圖表顯示。

Github

請至 Github repo 查看。
